# Bank Customer Churn Analysis (Power BI)

## Project Overview

This project analyzes customer churn behavior in a banking dataset using Power BI. The goal is to identify key factors contributing to customer attrition and provide actionable business insights.

## Objectives

* Measure overall churn rate
* Identify high-risk customer segments
* Analyze patterns based on demographics and account activity
* Support retention strategy decisions

## Dataset

The dataset includes:

* Customer demographics (Age, Gender, Geography)
* Account information (Balance, Tenure, Credit Score)
* Activity indicators (Active status, products used)
* Churn status (Exited)

## Key Insights

* Customers with low account balance showed higher churn rates
* Certain regions had significantly higher churn
* Inactive customers were more likely to leave
* Mid-tenure customers showed unexpected churn spikes
